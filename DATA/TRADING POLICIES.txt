# Candle Brain — Active Entry Policies

Pipeline: Detect → Bible (RAM) → ML cost-aware → Fire

## Active pattern families
1. Engulfing / Piercing / Dark Cloud
2. Pin bar / Hammer / Shooting Star
3. Morning / Evening Star
4. Inside-bar break (Bible)
5. Harami / Tweezers / Doji extremes
6. Three White / Three Black
7. Belt hold / Marubozu continuation

## Filters
- EMA50 / EMA200 trend + local slope (tight 0.10%)
- Engulfing: full body cover (Nison) + with-trend preferred; counter-trend only on exhaustion
- No impulse chase: block BUY/SELL when price stretched far above/below EMA8+EMA21 (PDF overbought)
- THREE_WHITE/BLACK only after pullback / near MA (buy start of impulse, not tip)
- Max 1 same-side auto position per pair (no stacking into a spike)
- Candle gap OFF (0 bars between entries)
- HARD: volume ≥ 3× Vol MA + rel_vol ≥ 3× rel_candle size + vs prev ≥ 2× — else NO TRADE
- Min pattern strength ≥ 1.5
- Same-bar bull+bear conflict = NO_TRADE
- ML cost-aware: λ=3.0 × fee hurdle + abs candle range ≥ 0.06%

## Exits
Profit lock +0.20% floor / 1.5× trail from peak / hard +1.2%.
Stop-loss (reverse of profit rules): arm at −0.20% / 1.5× trail from trough / hard −1.2%.
