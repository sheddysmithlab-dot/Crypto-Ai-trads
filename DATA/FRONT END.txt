Mobile View (Light Mode/Compact): Mobile par design vertical aur clean hai, jahan sirf sabse zaruri elements (Chart, Portfolio Summary, aur Trades) cards ke format me highlight kiye gaye hain.

Portfolio Tracking: Top bar me Total Capital ($142,560.88), Daily Profit, aur Open Positions dikh rahi hain. Yahi woh section hai jo aapke 2.5% Emergency Stop-Loss logic ko continuously track karega.

Ek high-speed trading bot ke liye frontend ki wiring bahut strong aur lag-free honi chahiye. Iski summary ko hum 4 main hisson me baant sakte hain:

1. Live Data "Wires" (WebSocket Connections)
Crypto market me prices har second badalte hain, isliye normal refresh wala system yahan kaam nahi karega. Yahan WebSockets (ek continuous live pipe) ka use hoga:

Price Feed Wire: Bybit API se direct WebSocket connect hoga jo Chart aur Live Trades table me millisecond ki speed se current price update karega.

PnL & Portfolio Wire: Aapke backend se ek WebSocket connect hoga jo total capital aur 24h profit ko real-time me update karega. Yahi connection track karega ki kab loss 2.5% ke pas pahunch raha hai.

2. Command "Wires" (REST API Calls)
Jab aap screen par koi action lete hain, toh frontend wo command piche baithe AI Agent (Backend) ko bhejta hai:

START Action: Jab aap bot on karte hain, frontend ek command bhejta hai: POST /start-bot. Isse background me aapka trailing lock (0.07+ wala logic) activate ho jata hai.

Master Kill Switch: Niche jo bada laal "STOP TRADING" button hai, uski wiring sabse high-priority par hogi. Jaise hi aap isay dabayenge, yeh frontend se turant POST /emergency-exit trigger karega, jo Bybit API ko sari trades market price par turant sell karne ka order dega.

3. Component-Specific Connections (UI Elements)
TradingView Chart: Frontend me TradingView ki Lightweight Charts library wire ki jayegi. AI Agent is chart ko signals bhejega taki aapke active lock points (jaise 0.08, 0.09) aur entry points chart par horizontal lines ki tarah live dikhai dein.

Live Active Trades Table: Yeh table aapke backend database (ya memory) se wired hogi. Agar AI Agent trailing lock hit hone par (jaise 0.10 se 0.08 aane par) trade sell kar deta hai, toh frontend use turant is table se hata dega ya status 'Closed' kar dega.

Backend AI Logic Connection: Aapka frontend khud koi trading decision nahi lega (taaki app fast chale). Frontend sirf AI Agent (jo Python/Node.js me backend par chal raha hai) ke decisions ka "display monitor" ban kar rahega.

4. Safety & Reconnection (Error Wiring)
Auto-Reconnect: Agar aapka internet mobile ya laptop par ek second ke liye band hota hai, toh frontend me aisi wiring (logic) honi chahiye jo connection aate hi wapas Bybit aur backend se jud jaye, bina bot ko disturb kiye.

System Alert Wiring: Agar AI Agent khud 2.5% loss detect karke emergency exit leta hai, toh backend frontend ko signal bhejega aur UI par ek bada Pop-up (Alert) aayega: "Emergency Exit Triggered: Trades Closed".
