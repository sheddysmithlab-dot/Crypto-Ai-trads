# AI Trading Agent — System Role & Operations

**Candlestick Trading Bible:** loaded in RAM at backend startup
(`DATA/candlestick_bible_memory.json` → `candlestick_bible_memory.py`).
Fetch any section in ~microseconds via `fetch_bible(id_or_alias)` / API
`GET /agent/bible/fetch?q=hammer`. Do not re-read the PDF at runtime.

**Pattern detection engine:** `volume_spread_system.py` (rules TBD — currently
NO_TRADE until patterns are coded). Intro vocabulary: `CANDLESTICK_PATTERNS_INTRO.md`.

## Profit Book (Active — code)
- Activate lock at +0.15% gross per trade
- Lock steps every +0.02% from peak
- Sell when price retreats to previous lock step
- Never sell below +0.15% gross

## UI & Control
1. START — agent monitors and executes in background
2. Live chart + locks from Bybit feed
3. BUY = manual LONG | SELL = manual SHORT (automation OFF only)
