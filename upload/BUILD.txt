SIMPLE FRONTEND UPLOAD (File Manager)
=====================================

Backend API (HTTPS — port 8000 mat use karo browser se):
   https://srv1831231.hstgr.cloud

VPS par backend chalao:
   docker compose -f docker-compose.backend.yml up -d --build

1) PC par rebuild (API URL change ho to):
   cd frontend
   copy ..\\upload\\frontend\\.env.production .env.production
   npm run build

2) frontend\\dist\\ ki files → upload\\frontend\\public_html\\
   (ya seedha dist se File Manager public_html me upload)

3) Hostinger File Manager → aitrads.in → public_html → files upload

4) Browser: https://aitrads.in

Firewall: ports 80 + 443 open (8000 ki zaroorat nahi).
